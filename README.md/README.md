# Toronto Crime Analytics Dashboard

An end-to-end data analytics project built with Python and Power BI to analyze crime trends across Toronto neighbourhoods using publicly available data.

## Project Overview

This project analyzes Toronto crime data from 2014–2025 using Python for data preparation and Power BI for interactive visualization.

The raw dataset was cleaned, transformed, and reshaped into an analysis-ready format before being used to create an interactive dashboard.



## Technologies Used

- Python
- Pandas
- Power BI
- Git
- GitHub



## Dashboard Features

- Total Crimes KPI
- Crime Trends (2014–2025)
- Crime by Type
- Top 10 Neighbourhoods
- Interactive Year Filter
- Interactive Crime Type Filter


## Skills Demonstrated

- Data Cleaning
- Data Transformation
- Exploratory Data Analysis
- Data Visualization
